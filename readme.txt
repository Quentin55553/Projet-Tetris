Tout d'abord, le programme est séparé en 6 fichiers (Makefile non compris).

Avant d'exécuter le programme, il faut d'abord le compiler.
Pour cela, la commande "make exec" est à utiliser dans le terminal et permet de transformer tous les fichiers ".c" en fichiers objets (.o).

Pour exécuter le programme, il vous suffit d'utiliser la commande "./exec" dans le terminal à la suite de la compilation.

La commande "make clean" est à utiliser dans le terminal et permet de supprimer tous les fichiers objets (.o) ainsi que l'exécutable (exec).

Une fois le programme exécuter, vous n'avez plus qu'à suivre les instructions données dans le terminal.
